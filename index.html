Tutorial De Uso Real

1. Fazer O Pedido Do Dia
  1. Entre Na Aba Pedido.
  2. Escolha O Fornecedor, Se Quiser Ver Apenas Um Vendedor.
  3. Use A Lista Para Selecionar A Quantidade De Cada Produto.
  4. Use O Controle Meia Para Meia Caixa Ou Meia Duzia.
  5. Produto Selecionado Fica Com Codigo E Nome Em Vermelho.
  6. No Final Da Pagina, Use O Botao WhatsApp Do Fornecedor Para Enviar So Os Produtos Dele.
2. Organizar A Ordem Dos Produtos
  1. No Campo Ordem, Escolha Editor Principal Ou Favoritos.
  2. Toque Em Editar Ordem.
  3. Use As Setas Para Subir Ou Descer 1 Produto.
  4. Use As Setas Com 5 Para Subir Ou Descer 5 Posicoes.
  5. Toque Em Salvar Ordem.
  6. Escolha Se Vai Salvar Em Editor Principal Ou Favoritos.
  7. Confirme Para Gravar. Se Cancelar, A Ordem Volta Como Estava.
3. Receber Valores E Lancar Custo X Venda
  1. Depois Que O Vendedor Mandar Os Valores, Entre Em Lancar Custo X Venda.
  2. Digite O Custo Embalagem Em Cada Produto Ou Use Colar Mensagem Para Importar O Texto Do WhatsApp.
  3. No App Android, Use Ler Foto Para Transformar A Foto Em Texto E Conferir Os Valores.
  4. Confira Produto, Quantidade E Custo Antes De Aplicar.
  5. Se O Pedido For Meia Caixa Ou Meia Duzia, Digite O Valor Pago Pela Meia Embalagem.
  6. O App Calcula O Custo Por Kg Ou Unidade Corretamente.
  7. Use Markup % Ou Preco Venda Para Ajustar A Venda.
  8. Se O Preco Venda Ficar Abaixo Do Custo, O Markup Aparece Negativo.
4. Enviar Para Weber Sistemas
  1. Confira Se Todos Os Produtos Do Pedido Tem Codigo.
  2. Confira Se Todos Os Produtos Ja Tem Custo Lancado.
  3. Use Enviar Drive Padrao Para Gerar O TXT Do Layout Padrao Da Weber.
  4. Use Enviar Drive XML Weber Para Gerar Um XML MOD 99 Para Testar No Botao NFe-XML.
  5. O Numero Da Nota Do XML E Automatico E Nao Se Repete.
  6. Envie Ou Salve O Arquivo No Google Drive.
  7. No Weber, Importe Pelo Botao Correspondente Ao Arquivo Gerado.
5. Cadastros De Produtos E Fornecedores
  1. Entre Na Aba Cadastros.
  2. Cadastre Ou Edite Fornecedores Na Coluna Fornecedores.
  3. Para Criar Produto, Toque Em Novo Produto.
  4. Para Editar Produto Existente, Toque Em Editar No Produto Da Lista.
  5. O Editor Abre Abaixo Do Produto Escolhido.
  6. Informe Codigo, Nome, Embalagem, Quantidade, Unidade Base, Markup Padrao E Imagem.
  7. Toque Em Salvar Produto.
6. Backup E Restauracao
  1. Use Baixar Backup Para Salvar Todos Os Dados Importantes Do App.
  2. Esse Arquivo Guarda Cadastros, Produtos, Imagens Editadas, Pedido Aberto, Historico, Custos E Ordens.
  3. Guarde O Arquivo No Google Drive Ou Em Lugar Seguro.
  4. Use Restaurar Backup Para Voltar Esses Dados Em Outro Celular Ou Apos Reinstalar.
  5. A Restauracao Substitui Os Dados Atuais, Entao O App Sempre Pede Confirmacao.
